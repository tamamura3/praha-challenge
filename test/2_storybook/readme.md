# 課題3
Storybookのメリット
- アプリを動かさずにUIコンポーネントの作成や変更、確認ができる
- UIコンポーネントの管理ができる。Storybookがドキュメントの役割を果たす
Storybookのデメリット
- アプリとStorybookの両方を管理する必要がある
